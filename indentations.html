Indentations

Python does not depend on "{}" to indicate a block of code. It depends on indentations like we do in our writing classes when we write paragraphs and essays.

You indent by clicking the Tab key from your keyboard.

In Python, indentations easily indicate a block of code especially what is/are inside those conditional (if, elif, else) and looping (for, while) statements.

In the example below, as you see right here, it is clearly indicated that if 2 is higher than 1 then it will display that "I need to buy 2 pens."

if (2 > 1):

  print("I need to buy 2 pens.")

This is the output of the code.

I need to buy 2 pens.

Adapted from: Sullivan, W. (2018). Python Programming Illustrated for Beginners and Intermediates. Retrieved from http://python.healthypslife.com/

QUIZ
  1. Type if.
  2. Type a () on its right.
  3. Inside the (), type 10 > 5.
  4. Type a : on the right of the ().
  5. Press Enter then press Spacebar 3 times.
  6. Display "10 is greater than 5."
