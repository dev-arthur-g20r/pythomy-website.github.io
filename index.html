Novice

A novice is the student in this application. He/she learns how to code by reading the lessons and taking the quizzes via an online interpreter to finish all the lessons so he/she will fully understand Programming by using Python. He/she asks for help or helps others by commenting in the Comments Section, creating or participating in an open forum, or directly messaging fellow novices or experts. After earning large amounts of points or medals, he/she will be promoted as Expert wherein he/she will be paid someday while working as one of our experts.

Expert

An expert is the teacher in this application. He/she looks in the lessons (with unlimited access) to review the lessons or use its Comments Section by commenting to help the novices in need while they are learning here how to code using Python. They are also privileged to help novices by directly messaging them or participate in an open forum created by them, a user or a fellow expert. They will soon be paid money by earning points and medals as expert based on the their help and its evaluation by novices.

Python Syntax

Unlike other object-oriented programming languages, Python does not use semi-colon (;) to separate two lines of code.

As you see in the example, we have written 2 different lines of code without using semi-colon for ending the statement of each.

The print() method displays in the console (the software with only the black background and white text) inside your monitor screen the content inside its parentheses.

print("I love chocolates!")

print("Python is the best.")

Here are the results when you run the code.

I love chocolates!

Python is the best.

Displaying multiple values in one line

You can print 2 values in the same output line by simply putting , after the first variable or value inside print() then typing in the second variable or value, displaying them in one line separated by a space (" ")

a = "Hello"

b = "Earth"

print(a, b)

Hello Earth

Now display on your screen, "Hello World!"

QUIZ
  1. Type the word "print".
  2. Type () after the word print.
  3. Inside the (), type "".
  4. Inside the "", type Hello World!
