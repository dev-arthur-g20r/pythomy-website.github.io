Sets

Sets are collections that have no index and order. That is why when you display them, the items inside them are in any order.

It cannot have more than 1 item with a value same with another item(s).

This collection encloses items in {} (braces).

Like lists, sets can have items of any data type.

Accessing Set Items

Since sets are unordered and have no index, we cannot access it individually though we can access them though using for loop.

starterPokemon = {"Charmander", "Squirtle", "Bulbasaur"}


for pokemon in starterPokemon:

   print(pokemon)

Squirtle

Charmander

Bulbasaur

We can check if a data is an item in a set using in.

print("Bulbasaur" in starterPokemon)

print("Groudon" in starterPokemon)

True

False

set.add(item)

We use this method to add a single item in a set.

myFavBands = {"Linkin Park", "Nickelback", "Puddle of Mudd"}


myFavBands.add("Godsmack")

print(myFavBands)

{'Godsmack', 'Nickelback', 'Puddle of Mudd', 'Linkin Park'}

set.update(list)

We use this method to add multiple items in a set.

myFavBands.update(["Paramore", "Evanescence"])

print(myFavBands)

{'Godsmack', 'Puddle of Mudd', 'Nickelback', 'Paramore', 'Evanescence', 'Linkin Park'}

len(set)

We use this method to get the length of a set.

numberOfFavBands = len(myFavBands)

print(numberOfFavBands)

7

set.remove(item)

We use this method to remove an item from a set. It raises an error when item does not exist.

myFavBands.remove("Godsmack")

print(myFavBands)

{'Evanescence', 'Linkin Park', 'Paramore', 'Puddle of Mudd', 'Nickelback'}

set.discard(item)

We use this method to also remove an item from a set but unlike the remove() method, it does not raise an error when item does not exist.

myFavBands.discard("Nickelback")


print(myFavBands)

{'Linkin Park', 'Paramore', 'Evanescence', 'Puddle of Mudd'}

set.pop()

We use this method to remove the last item of a set.

myFavBands.clear()

set1.union(set2)

We use this method to return a new set containing all items from both sets.

starters = {"Torchic", "Mudkip", "Treecko"}

legends = {"Groudon", "Kyogre", "Rayquaza"}


pokemonSet = starters.union(legends)


print(pokemonSet)

{'Kyogre', 'Treecko', 'Groudon', 'Mudkip', 'Torchic', 'Rayquaza'}

| operator

We use | as an alternative for set1.union(set2).

starters = {"Torchic", "Mudkip", "Treecko"}

legends = {"Groudon", "Kyogre", "Rayquaza"}


pokemonSet = starters | legends


print(pokemonSet)

{'Kyogre', 'Torchic', 'Rayquaza', 'Treecko', 'Groudon', 'Mudkip'}

set1.update(set2)

We use this method to insert all items of set2 to set1.

sweets = {"Reese's", "Hershey's", "M&M's"}

moreSweets = {"Toblerone", "Maltesers"}

print(sweets)

{"Hershey's", 'Maltesers', 'Toblerone', "Reese's", "M&M's"}

set1.intersection(set2)

We use this method to return a new set containing all items both set1 and set2 have.

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


colorsInCommon = hisColors.intersection(herColors)


print(colorsInCommon)

{'Red'}

& operator

We use & as an alternative for set1.intersection(set2).

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


colorsInCommon = hisColors.intersection(herColors)


print(colorsInCommon)

{'Red'}

set1.difference(set2)

We use this method return a new set containing all items only set1 has that set2 does not have.

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


colorsOnlyHeHas = hisColors.difference(herColors)


print(colorsOnlyHeHas)

{'Blue', 'Yellow'}

- operator for sets

We use - as an alternative for set1.difference(set2).

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


colorsOnlySheHas = herColors - hisColors


print(colorsOnlySheHas)

{'Black', 'Green'}

set1.symmetric_difference(set2)

We use this method to return a new set containing all items of set1 and set2 except what they both have in common.

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


differentColors = hisColors.symmetric_difference(herColors)


print(differentColors)

{'Black', 'Blue', 'Green', 'Yellow'}

^ operator

We use ^ as an alternative for set1.symmetric_difference(set2).

hisColors = {"Red", "Blue", "Yellow"}

herColors = {"Black", "Red", "Green"}


differentColors = hisColors ^ herColors


print(differentColors)

{'Green', 'Black', 'Yellow', 'Blue'}

set(collection)

We use this casting method to convert a list or tuple to a set.

songs = ["In The End", "Numb", "One Step Closer"]

linkinParkSongs = set(songs)

print(linkinParkSongs)

{'One Step Closer', 'In The End', 'Numb'}
